# Fabric App - ETL config

Una aplicación de Fabric creada a partir de la plantilla *Basic Todo App* de *Rayfin*, que gestiona una tabla de configuración de una ETL. 

Para implementar la tabla en el frontend utiliza la biblioteca [TanStack Table](https://tanstack.com/table/latest).
